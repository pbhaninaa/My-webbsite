High School Application System

This is a school admission system that facilitates the application process for both learners and administrators. Learners can search for schools they are interested in and apply online by first registering with the system. Administrators, on the other hand, can log in and access the applications received, then process them accordingly. The system was developed using AngularJS for the frontend and Spring Boot with Java for the backend, providing a robust and reliable foundation for the application. By leveraging these technologies, the system can deliver a fast and seamless user experience, with intuitive and user-friendly interfaces for both learners and administrators. This ensures that learners can easily find the schools they want and apply with minimal hassle, while administrators can efficiently manage the applications they receive.
